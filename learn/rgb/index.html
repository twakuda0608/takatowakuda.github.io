RGB とは？

画面の色は 赤（Red）・緑（Green）・青（Blue） の 3 色の光を混ぜて作られます。 それぞれ 0〜255 の 256 段階で強さを指定でき、組み合わせると 2563 = 約 1677 万色 を表現できます。

黄 (R+G)
R:255 G:255 B:0
マゼンタ (R+B)
R:255 G:0 B:255
シアン (G+B)
R:0 G:255 B:255
白 (全部)
R:255 G:255 B:255
黒 (なし)
R:0 G:0 B:0

光の混色（加法混色）

光は重ねるほど明るくなります。赤・緑・青を全部重ねると白になります。 スクリーンやプロジェクターはこの原理で色を表示しています。

インク・絵の具の混色（減法混色）

インクや絵の具は重ねるほど暗くなります。シアン・マゼンタ・イエロー（CMY）を全部重ねると黒になります。 プリンターのインクはこの原理を使っています（光の三原色 RGB とは異なります）。
